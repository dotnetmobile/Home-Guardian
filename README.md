# Home-Guardian
  by dotnetmobile@gmail.com
---

Proof of concept taking a picture of an intruder when very close (&lt;=10 cm) to the home.

* Raspberry pi GPIO
     https://www.raspberrypi.org/documentation/usage/gpio/
